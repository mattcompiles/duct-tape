# duct-tape

Duct-tape is a JS/TS module bundler written in Rust. It uses [SWC](https://swc.rs/) under the hood for parsing and transforms.

## Features

| Feature    | Status                | Notes |
| ---------- | --------------------- | ----- |
| ES modules | Partially implemented |       |
| CommonJS   | WIP                   |       |
| JSX        | Soon                  |       |
| TypeScript | Soon                  |       |
